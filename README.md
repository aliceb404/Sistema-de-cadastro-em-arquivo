# Sistema de cadastro em arquivo

O presente projeto foi proposto com fins acadêmicos, no qual o programa deve ler, inserir e remover itens em um banco de dados com um total 100 elementos cadastrados.

A base de dados foi criada pelos autores (podendo ser dados não 
embasados), a extensão arquivo utilizado no projeto foi o .csv, o tema campos escolhidos para elaborar este projeto foram, respectivamente, **Países: Identificador (int); Nome (getline); População (int); Idioma (getline); Continente (getline)**. Além disso, o programa deve retornar, na tela, um registro através da chamada de somente um dos campos registrados ou, também, a lista 
inteira, todas as possibilidades devem estar disponíveis para o usuário. Sucintamente, o 
projeto objetiva desenvolver um sistema de registro no qual o usuário possa remover, 
adicionar e consultar uma informação em um banco de dados, o programa deve devolver 
a informação inteiramente ou pela categoria selecionada.
